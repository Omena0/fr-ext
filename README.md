
# Frscript support

Adds [Frscript](https://github.com/Omena0/fr) language support to VSCode.

## Installing fr

```zsh
pip install fr
```

## Features

- Syntax highlights
- Debugging
- Autocomplete
- Python integration
- Basic type checker
- Rename symbol
- Refactorings
- Much more that i cant be bothered to write
